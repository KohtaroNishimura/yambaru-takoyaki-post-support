# やんばる あちこーこー たこ焼き 投稿サポートアプリ

「やんばるの日常」を、毎日の発信に。
沖縄・名護の屋台文化や“あちこーこー（熱々）”の空気感を活かしながら、
毎朝のSNS投稿をサポートする、屋台運営向けの投稿支援アプリです。

このアプリは単なる文章生成ではなく、

* 地域密着
* 焼きたて感
* 営業情報
* 天候や季節感
* 常連との距離感

を自然に発信し、
「また今日も行こうかな」と思ってもらえる日常導線を作ることを目的としています。

---

# コンセプト

## 「やんばるで、あちこーこーの嬉しさを。」

スーパー前で漂うソースの香り。
学校帰りの子どもたち。
夕方の買い物帰り。
“今日はたこ焼きあるかな？”という地域の日常。

沖縄北部・やんばるの日常風景に溶け込みながら、
焼きたての嬉しさを毎日の投稿として届けます。 

---

# このアプリで解決したいこと

屋台営業は想像以上に忙しく、

* 投稿を考える時間がない
* 営業直前まで仕込みで手が離せない
* 毎回同じ投稿になりがち
* 告知したいけど文章が思いつかない

という課題があります。

特にワンオペ営業では、
「投稿を考える時間」自体が負担になります。 

そこで、

> “毎朝1投稿だけ、自然に作れる”

ことを最優先に設計します。

# 主な機能

## 1. 今日の営業投稿を自動生成

毎朝の投稿生成に必要な情報を取得します。

取得情報：

* 今日の日付
* 曜日
* 天気
* 気温
* 旧暦
* 六曜
* 二十四節気
* 日の出 / 日の入
* 潮汐情報

固定情報：

* 出店場所（JAファーマーズ前）
* 営業時間（暫定）

  * 12:00〜17:30

※天候・仕込み状況・イベント出店等により変更の可能性があります。

---

## 2. 屋台営業向けテンプレート

屋台・移動販売に特化した文章生成を行います。

反映する要素：

* 焼きたて感（あちこーこー）
* 地域密着感
* 買い物導線
* 子育て世帯への親和性
* 季節感
* 沖縄らしい言い回し
* ウチナーグチ
* 名護・やんばる感

---

## 3. 投稿スタイル切り替え

以下のような投稿トーンを選択可能。

* 朝営業告知
* 完売報告
* お礼投稿
* 雨の日投稿
* 子ども・家族エピソード
* イベント出店告知
* 常連向け投稿
* “今日の一言”系

---

## 4. SNS向け最適化

X（旧Twitter）向けに、

* 文字数
* 改行
* 読みやすさ
* ハッシュタグ
* 感情温度

を調整します。

# 実装済みMVP（このリポジトリ）

READMEの要件をもとに、以下を実装しました。

- 今日の情報取得（天気 / 気温 / 日の出日の入 / 日付）
- 朝投稿（1種類）生成
- コピーボタン
- 固定営業情報（JAファーマーズ前 / 12:00〜17:30）

補足：
- 旧暦 / 六曜 / 二十四節気は `cnlunar` ライブラリ連携で取得
- 潮汐は `astral` の月齢データから潮名（大潮/中潮/小潮/長潮/若潮）を算出
- 投稿文に沖縄らしい小ネタを1行差し込む
- `OPENAI_API_KEY` を設定するとOpenAI生成を優先し、未設定時はローカルテンプレート生成に自動フォールバック

## 起動手順

### 1. Backend（FastAPI）

```bash
cd backend
python3.13 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --reload --port 8000
```

### 2. Frontend（Next.js）

```bash
cd frontend
npm install
cp .env.local.example .env.local
npm run dev
```

### 3. アクセス

- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:8000`
- ヘルスチェック: `http://localhost:8000/health`

## API概要

- `GET /health`
- `GET /today`
- `POST /generate`
